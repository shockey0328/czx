# 橙子学数据看板 - 快速启动指南

## 🎯 最快启动方式

### Windows用户（推荐）

**双击运行以下任一文件：**
- `start-server.bat` 
- `start-server.ps1`

浏览器会自动打开 http://localhost:8000/dashboard.html

---

## 📖 详细启动步骤

### 方法一：使用启动脚本（最简单）

1. 找到项目根目录
2. 双击 `start-server.bat` 或 `start-server.ps1`
3. 等待浏览器自动打开
4. 开始使用看板！

### 方法二：使用Python（推荐）

```bash
# 1. 打开命令行/终端
# 2. 进入项目目录
cd 橙子学数据看板

# 3. 启动服务器
python -m http.server 8000

# 4. 在浏览器中访问
# http://localhost:8000
```

### 方法三：使用Node.js

```bash
# 1. 打开命令行/终端
# 2. 进入项目目录
cd 橙子学数据看板

# 3. 启动服务器
npx http-server -p 8000

# 4. 在浏览器中访问
# http://localhost:8000
```

### 方法四：使用VS Code

1. 安装 **Live Server** 扩展
2. 右键点击 `dashboard.html`
3. 选择 **"Open with Live Server"**
4. 自动在浏览器中打开

## 🎨 使用看板

### 切换不同的看板

**方式一：使用筛选器**
1. 点击顶部的"周度"或"月度"按钮切换时间维度
2. 使用下拉选择器选择具体的看板

**方式二：键盘快捷键**
- `Alt + W`：切换到周度
- `Alt + M`：切换到月度
- `Alt + 左/右箭头`：在当前周期内切换看板
- `Alt + 1/2/3/4`：快速切换到指定看板

### 各看板功能说明

#### 1️⃣ 核心数据看板（周度）
- 查看周度核心指标
- 分析用户活跃和留存
- 监控营收和ARPU
- AI智能分析

#### 2️⃣ 搜索数据看板（周度）
- 热搜词TOP 100
- 词云可视化
- 搜索行为漏斗
- 转化率趋势
- 留存分析

#### 3️⃣ 核心数据看板（月度）
- 月度核心指标
- 长期趋势分析
- 同比环比对比

#### 4️⃣ 模块渗透率看板（月度）
- 各模块渗透率
- 趋势和排名
- 热力图分析
- AI优化建议

## ⚠️ 常见问题

### Q: 为什么不能直接双击HTML文件打开？
**A:** 由于浏览器安全限制，需要通过HTTP服务器访问才能正常加载数据文件。

### Q: 启动脚本提示"未检测到Python或Node.js"？
**A:** 需要先安装Python或Node.js：
- Python下载：https://www.python.org/downloads/
- Node.js下载：https://nodejs.org/

### Q: 看板显示空白或数据加载失败？
**A:** 请检查：
1. 是否通过HTTP服务器访问（地址栏显示 http://）
2. CSV数据文件是否存在
3. 浏览器控制台（F12）是否有错误信息

### Q: 切换看板时很慢？
**A:** 首次加载每个看板时需要一些时间，之后会自动缓存，切换会更快。

### Q: 如何停止服务器？
**A:** 在命令行窗口按 `Ctrl + C`

## 💡 使用技巧

1. **书签收藏**：将 http://localhost:8000/dashboard.html 添加到浏览器书签
2. **全屏查看**：按 F11 进入全屏模式，获得更好的查看体验
3. **多屏显示**：可以在不同浏览器标签页打开不同看板，方便对比
4. **定期刷新**：数据更新后，刷新页面（F5）即可看到最新数据

## 📱 移动端访问

### 同一WiFi网络下访问

1. 启动服务器后，查看电脑的IP地址
   ```bash
   # Windows
   ipconfig
   
   # Mac/Linux
   ifconfig
   ```

2. 在手机浏览器中访问：
   ```
   http://[电脑IP地址]:8000/dashboard.html
   例如：http://192.168.1.100:8000/dashboard.html
   ```
