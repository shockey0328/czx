如果GitHub推送一直失败，可以使用Railway CLI直接部署：

1. **安装Railway CLI**
   ```bash
   npm install -g @railway/cli
   ```

2. **登录Railway**
   ```bash
   railway login
   ```

3. **链接项目**
   ```bash
   railway link
   ```

4. **直接部署**
   ```bash
   railway up
   ```
