## 💡 使用指南

### 基础操作

1. **选择月份**
   - 使用顶部下拉菜单选择月份
   - 所有数据自动更新

2. **查看指标**
   - 核心指标卡片显示当前值
   - 环比：与上月对比
   - 同比：与去年同月对比

3. **切换时间范围**
   - 点击趋势图上方的按钮
   - 选择3/6/12个月
   - 图表实时更新

4. **查看B端数据**
   - 滚动到页面中部
   - 查看6个B端指标

### AI功能使用

1. **AI数据分析**
   - 自动生成，无需操作
   - 点击"刷新分析"重新生成
   - 查看三列分析内容

2. **小橙子AI助手**
   - 点击页面左侧的小橙子图标
   - 使用快捷问题或输入自定义问题
   - 查看小橙子的回答
   - 点击×关闭对话窗口

- "分析本月数据"

## ⚠️ 注意事项

### 使用建议
1. **网络连接** - AI功能需要网络访问DeepSeek API
2. **API限制** - 注意调用频率，避免过度使用
3. **数据隐私** - 仅发送统计数据，不含敏感信息
4. **浏览器** - 推荐Chrome或Edge最新版

### 常见问题

**Q: AI分析不显示？**
A: 检查网络连接，或等待自动切换到本地分析

**Q: 小橙子不回答？**
A: 检查网络连接，或稍后重试

**Q: 同比数据为空？**
A: 需要有去年同月的数据（当前有13个月数据）

**Q: 如何更新数据？**
A: 运行 `更新数据.bat`，然后刷新页面
