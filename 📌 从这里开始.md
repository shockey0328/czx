# 📌 从这里开始

欢迎使用橙子学数据看板！这是一个快速导航页面。

---

## 🎯 我想...

### 1️⃣ 立即查看看板（本地测试）

**→ 双击这个文件**: `启动本地服务器.bat`

或者

**→ 右键这个文件**: `启动本地服务器.ps1` → 选择"使用 PowerShell 运行"

浏览器会自动打开，你就能看到看板了！

---

### 2️⃣ 部署到线上（让别人也能访问）

**→ 打开命令行**，依次输入：

```bash
npm install -g vercel
vercel login
vercel --prod
```

完成后会得到一个网址，分享给团队即可！

详细步骤请看：`快速上手指南.md`

---

### 3️⃣ 了解项目详情

**→ 阅读这些文档**：

| 文档 | 内容 |
|------|------|
| `快速上手指南.md` | 🚀 最简单的入门教程 |
| `完成总结.md` | ✅ 项目完成情况和功能说明 |
| `当前状态说明.md` | 📋 详细的技术文档和配置 |
| `README.md` | 📖 完整的使用手册 |

---

### 4️⃣ 测试各个看板是否正常

**→ 打开这个文件**: `test-local.html`

点击各个"加载看板"按钮，查看是否正常显示。

---

### 5️⃣ 更新数据

**→ 找到对应的 CSV 文件**，直接替换即可：

- 搜索数据: `搜索数据看板（周度）/*.csv`
- 渗透率数据: `各模块渗透率看板（月度）/各模块渗透率.csv`
- 核心数据: `核心数据看板（周度）/*.csv`

## 🆘 遇到问题？

### 问题1: 启动脚本没反应
→ 安装 Python 或 Node.js
- Python: https://www.python.org/downloads/
- Node.js: https://nodejs.org/

### 问题2: 看板显示空白
→ 按 F12 查看控制台错误信息

### 问题3: 无法部署
→ 检查网络连接，确认已安装 Node.js

### 问题4: 其他问题
→ 查看 `完成总结.md` 的"故障排除"部分
