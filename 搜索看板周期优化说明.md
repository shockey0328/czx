# 搜索数据看板周期选择优化

## 修改内容

### 1. 周期选择器改为倒序
**修改前：**
```
2026年第1周
2026年第2周
...
2026年第8周
```

**修改后：**
```
2026年第8周 ← 最新
2026年第7周
...
2026年第1周
```

### 2. 默认选择最新一周
- 页面打开时自动选择第8周（最新一周）
- 下拉菜单默认显示"2026年第8周"

## 修改的文件

### 1. `搜索数据看板（周度）/index.html`
```html
<select class="week-selector" id="weekSelector">
    <option value="8" selected>2026年第8周</option>  <!-- 添加 selected -->
    <option value="7">2026年第7周</option>
    <option value="6">2026年第6周</option>
    <option value="5">2026年第5周</option>
    <option value="4">2026年第4周</option>
    <option value="3">2026年第3周</option>
    <option value="2">2026年第2周</option>
    <option value="1">2026年第1周</option>
</select>
```

### 2. `搜索数据看板（周度）/app.js`
```javascript
let currentWeek = 8; // 从 1 改为 8
```

## 效果

✅ **打开看板时**：
- 自动显示第8周（最新一周）的数据
- 选择器默认显示"2026年第8周"

✅ **下拉选择时**：
- 从上到下：第8周 → 第7周 → ... → 第1周
- 符合时间倒序的习惯（最新的在最上面）

## 测试方法

1. 打开 `搜索数据看板（周度）/index.html`
2. 确认页面显示的是第8周的数据
3. 点击周期选择器，确认顺序为倒序（8→1）
4. 切换到其他周，确认数据正常更新

1. 在HTML中添加新选项到最上面
2. 修改 `currentWeek` 的默认值为最新周数
3. 保持倒序排列
