# 统一看板样式更新说明 v2

## 更新时间
2026年2月27日

## 设计参考
参考原有月度数据看板的设计风格（橙子学数据看板），保持品牌一致性和视觉统一。

## 核心设计元素

### 1. 配色方案
- **主色调**：`#ff7043`（橙色）- 品牌色
- **背景色**：`#f5f5f5`（浅灰）- 干净简洁
- **文字色**：`#333`（深灰）- 易读性好
- **边框色**：`#e0e0e0`（浅灰）- 轻量分隔
- **悬停色**：`#fff5f0`（浅橙）- 温暖友好

### 2. 设计风格特点
- **简洁清爽**：白色导航栏 + 浅灰背景
- **卡片式布局**：圆角、阴影、留白
- **橙色点缀**：按钮、链接、强调元素
- **轻量边框**：1px细线，不喧宾夺主

## 详细设计规范

### 导航栏设计
```css
背景：白色 (white)
阴影：0 2px 4px rgba(0, 0, 0, 0.08)
内边距：16px 32px
高度：约72px
```

**特点**：
- 轻量阴影，不压抑
- 适中的内边距
- Logo + 标题 + 控件的经典布局

### Logo和标题
```css
Logo高度：40px
标题字号：18px
标题粗细：400 (normal)
标题颜色：#333
```

**设计理念**：
- Logo适中，不过大
- 标题使用常规字重，不过于突出
- 黑色文字，专业稳重

### 周度/月度切换按钮
```css
背景：白色
边框：1px solid #e0e0e0
圆角：4px
内边距：6px 16px
字号：14px
```

**状态设计**：
- **默认**：白底灰边，灰色文字
- **悬停**：浅橙背景，橙色边框和文字
- **激活**：橙色背景，白色文字

**交互**：
- 快速过渡（0.2s）
- 清晰的状态反馈

### 看板类型选择器
```css
背景：白色
边框：1px solid #e0e0e0
圆角：4px
内边距：6px 32px 6px 12px
最小宽度：160px
```

**特色功能**：
- 自定义下拉箭头（SVG）
- 悬停时橙色边框
- 聚焦时外发光效果

### 加载动画
```css
尺寸：48px
边框：4px
颜色：#ffe0d6 / #ff7043
```

**设计**：
- 橙色系配色
- 适中的尺寸
- 流畅的旋转动画
