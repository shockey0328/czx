# 快速参考

## 🚀 快速启动

### Windows用户
```bash
# 双击运行
启动看板.bat
```

### 其他用户
```bash
# 终端1：启动主看板
python -m http.server 8000

# 终端2：启动React应用（可选）
cd 核心数据看板（月度）
npm install  # 首次运行
npm start
```

## 🔍 故障排查

### 问题：看板无法加载
```
1. 检查文件路径是否正确
2. 确认服务器已启动
3. 清除浏览器缓存
4. 查看浏览器控制台错误
```

### 问题：React应用无法访问
```
1. 确认已执行 npm install
2. 确认已执行 npm start
3. 检查端口3000是否被占用
4. 查看终端错误信息
```

### 问题：数据不显示
```
1. 检查CSV文件是否存在
2. 检查CSV文件格式是否正确
3. 查看浏览器控制台错误
4. 尝试刷新页面
```

## 💡 使用技巧

### 技巧1：快速切换看板
使用键盘快捷键（如果支持）或收藏常用看板

### 技巧2：数据更新
直接替换CSV文件，刷新页面即可看到最新数据

### 技巧3：性能优化
- 关闭不需要的浏览器标签
- 定期清理浏览器缓存
- 使用Chrome浏览器获得最佳性能

### 技巧4：多屏显示
在不同浏览器窗口打开不同看板，实现多屏对比

### 技巧5：导出数据
使用浏览器的打印功能或截图工具保存看板快照

## 🔐 安全提示

- 不要在公共网络上暴露本地服务器
- 定期更新依赖包
- 不要提交敏感数据到版本控制
- 使用HTTPS部署生产环境
