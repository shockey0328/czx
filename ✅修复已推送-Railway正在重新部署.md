## 💡 如果部署失败

### 查看错误日志

1. 登录Railway控制台
2. 查看部署日志
3. 找到错误信息

### 常见问题

**问题1：依赖安装失败**
```
npm ERR! ...
```
**解决：** 检查package.json中的依赖是否正确

**问题2：启动命令失败**
```
Error: Cannot find module 'xxx'
```
**解决：** 确认所有依赖都已安装

**问题3：端口配置错误**
```
Error: listen EADDRINUSE
```
**解决：** 确认使用环境变量PORT

**问题4：数据文件找不到**
```
Error: ENOENT: no such file or directory
```
**解决：** 检查Git LFS文件是否正确下载
