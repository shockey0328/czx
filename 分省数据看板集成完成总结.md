## 📊 看板功能概览

### 核心功能
1. **核心指标展示**
   - 各省月活、营收、ARPU等关键指标
   - 卡片式布局，数据清晰

2. **省份排名**
   - 活跃用户排名
   - 营收排名
   - ARPU排名
   - 使用率排名

3. **趋势分析**
   - 支持7个省份选择
   - 支持3/6/12个月周期
   - 三个趋势图表（月活、营收、使用率）

4. **AI数据分析**
   - 一键生成分析报告
   - 自动识别数据特点
   - 提供优化建议

### 数据支持
- 7个省份：福建、山东、江苏、广东、安徽、辽宁、河南
- 核心数据：分省核心数据.csv
- 趋势数据：每个省份独立的CSV文件

## 🔧 技术实现

### 1. 导航配置（main.js）
```javascript
'province-monthly': {
    name: '分省数据',
    path: '分省数据看板（月度）/index.html',
    type: 'static'
}
```

### 2. Chart.js本地化（index.html）
```html
<!-- 原来 -->
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

<!-- 现在 -->
<script src="../libs/chart.min.js"></script>
<script src="../libs/chartjs-adapter-date-fns.bundle.min.js"></script>
```

### 3. 文件结构
```
分省数据看板（月度）/
├── index.html              # 主页面
├── app.js                  # 主逻辑
├── app_new.js              # 备用逻辑
├── style.css               # 样式文件
├── 分省核心数据.csv        # 核心数据
└── 分省趋势数据（各省）.csv # 趋势数据
```
