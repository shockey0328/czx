# 🚀 立即开始使用橙子学数据看板

## 快速启动（3步）

### 步骤1：运行诊断脚本
双击运行以下文件之一：
- **诊断并修复.bat** （推荐，适合Windows）
- **诊断并修复.ps1** （PowerShell版本，功能更强）

### 步骤2：查看测试结果
脚本会自动打开测试页面，显示：
- ✅ HTTP服务器状态
- ✅ CSV文件访问测试
- ✅ 看板页面加载测试

### 步骤3：开始使用
如果所有测试都通过（显示✅），您可以：
- 访问主看板：`http://localhost:8000`
- 或继续使用测试页面中的链接

## 如果遇到问题

### 问题1：测试显示❌
**查看错误信息**，通常是以下原因之一：

1. **未安装Python或Node.js**
   - 下载Python: https://www.python.org/downloads/
   - 或下载Node.js: https://nodejs.org/

2. **端口被占用**
   - 脚本会自动尝试释放端口
   - 或手动关闭占用8000端口的程序

3. **文件缺失**
   - 检查是否完整解压了所有文件
   - 确保在正确的目录运行脚本

### 问题2：浏览器显示"数据加载失败"
**确认访问方式：**
- ✅ 正确：`http://localhost:8000`
- ❌ 错误：`file:///C:/...`（直接打开HTML文件）

**解决方法：**
- 必须通过HTTP服务器访问
- 运行 `诊断并修复.bat` 自动启动服务器

### 问题3：搜索数据看板没有数据
**运行测试：**
1. 访问：`http://localhost:8000/测试搜索看板.html`
2. 查看CSV文件测试结果
3. 如果显示❌，检查CSV文件是否存在

## 常用命令

### 启动服务器
```bash
# Python
python -m http.server 8000

# Node.js
npx http-server -p 8000
```

### 检查端口
```bash
# Windows
netstat -ano | findstr :8000

# 结束进程
taskkill /PID <进程ID> /F
```

### 测试文件访问
在浏览器中访问：
```
http://localhost:8000/搜索数据看板（周度）/第1周搜索词.csv
```
如果能下载文件，说明服务器正常。
