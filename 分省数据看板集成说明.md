# 分省数据看板集成说明

## ✅ 集成完成

分省数据看板已成功集成到橙子学数据看板系统的月度看板中。

## 📊 看板位置

**访问路径**: 主导航 → 月度 → 分省数据

```
橙子学数据看板
├── 周度看板
│   ├── 核心数据
│   └── 搜索数据
└── 月度看板
    ├── 核心数据
    ├── 各模块渗透率
    └── 分省数据 ✨ 新增
```

## 🎯 看板功能

### 1. 核心指标模块
展示各省的关键业务指标：
- 月活用户数
- 营收金额
- ARPU值
- 使用率
- 留存率等

### 2. 省份排名模块
四个维度的可视化排名：
- 活跃用户排名
- 营收排名
- ARPU排名
- 使用率排名

### 3. 趋势分析模块
- 支持选择省份（福建、山东、江苏、广东、安徽、辽宁、河南）
- 支持选择时间周期（近3个月、近6个月、近12个月）
- 三个趋势图表：
  - 月活趋势
  - 营收趋势
  - 使用率趋势

### 4. AI数据分析模块
- 一键生成分析报告
- 自动分析各省数据特点
- 提供可执行的优化建议

## 🔧 技术更新

### Chart.js引用更新
```html
<!-- 原来：CDN引用 -->
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

<!-- 现在：本地引用 -->
<script src="../libs/chart.min.js"></script>
<script src="../libs/chartjs-adapter-date-fns.bundle.min.js"></script>
```

**更新原因**：
- 确保离线环境下可以正常使用
- 提高加载速度和稳定性
- 与其他看板保持一致

## 🚀 使用方法

### 方法1：通过主导航访问
1. 打开 `index.html`（主导航页面）
2. 点击顶部的"月度"按钮
3. 在下拉菜单中选择"分省数据"
4. 看板将在iframe中加载

### 方法2：直接访问
1. 运行 `启动分省数据看板.bat`
2. 或直接打开 `分省数据看板（月度）/index.html`

### 方法3：使用测试页面
1. 打开 `测试分省数据看板集成.html`
2. 点击相应的测试按钮

## 📊 数据说明

### 核心数据文件
`分省核心数据.csv` 包含各省的核心指标：
- 省份

## 🔄 数据更新流程

### 更新核心数据

## 🔍 故障排查

### 看板无法加载
1. 检查文件路径是否正确
2. 确认CSV数据文件存在
3. 查看浏览器控制台错误信息

### 图表不显示
1. 确认Chart.js库已正确加载
2. 检查CSV数据格式是否正确
3. 清除浏览器缓存后重试

### 数据不更新
1. 确认CSV文件已保存
2. 强制刷新浏览器（Ctrl+F5）
3. 检查CSV文件编码（应为UTF-8）
