# 学科网智书数据看板 - 使用说明

## 📋 概述

这是一个整合了4个数据看板的统一平台，通过顶部的筛选器可以快速切换不同维度和类型的数据看板。

## 🎯 界面说明

### 顶部导航栏

```
┌─────────────────────────────────────────────────────────────┐
│  [Logo] 学科网智书数据看板    [周度] [月度]  [核心数据 ▼]  │
└─────────────────────────────────────────────────────────────┘
```

1. **Logo和标题**：左侧显示品牌标识
2. **周度/月度切换**：中间的按钮用于切换时间维度
3. **看板选择器**：右侧下拉框用于选择具体的看板

## 🔄 切换方式

### 方式一：鼠标操作

1. **切换时间维度**
   - 点击"周度"按钮：查看周度数据
   - 点击"月度"按钮：查看月度数据

2. **选择具体看板**
   - 点击下拉选择器
   - 从列表中选择要查看的看板

### 方式二：键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt + W` | 切换到周度 |
| `Alt + M` | 切换到月度 |
| `Alt + ←` | 切换到上一个看板 |
| `Alt + →` | 切换到下一个看板 |
| `Alt + 1` | 快速切换到第1个看板 |
| `Alt + 2` | 快速切换到第2个看板 |
| `Alt + 3` | 快速切换到第3个看板 |
| `Alt + 4` | 快速切换到第4个看板 |

## 📊 看板说明

### 周度看板

#### 1. 核心数据（周度）
- **功能**：展示周度核心业务指标
- **内容**：
  - 活跃用户、留存率、营收等核心指标
  - B端核心数据
  - 趋势分析图表
  - AI智能数据分析

#### 2. 搜索数据（周度）
- **功能**：展示搜索功能的使用数据
- **内容**：
  - 热搜词TOP 100排行
  - 词云可视化
  - 搜索行为漏斗
  - 搜索转化率趋势
  - 搜索功能留存分析

### 月度看板

#### 3. 核心数据（月度）
- **功能**：展示月度核心业务数据
- **内容**：
  - 月度核心指标
  - 长期趋势分析
  - 同比环比数据对比

#### 4. 各模块渗透率（月度）
- **功能**：展示各功能模块的渗透率
- **内容**：
  - 各模块渗透率趋势
  - 模块排名和分布
  - 热力图分析
  - AI智能分析和优化建议

## 💡 使用技巧

### 1. 快速访问
- 将页面添加到浏览器书签，方便下次访问
- 使用键盘快捷键提高操作效率

### 2. 数据对比
- 可以在不同浏览器标签页打开相同页面
- 分别选择不同的看板进行对比分析

### 3. 全屏查看
- 按 `F11` 进入全屏模式
- 获得更大的数据展示空间

### 4. 移动端访问
- 在同一WiFi网络下
- 使用手机浏览器访问电脑的IP地址
- 例如：`http://192.168.1.100:8000/dashboard.html`

### 5. 数据刷新
- 当数据更新后，按 `F5` 或 `Ctrl + R` 刷新页面
- 即可看到最新的数据

## ⚙️ 自动记忆功能

系统会自动记住你上次查看的看板：
- 下次打开页面时，会自动显示上次查看的看板
- 无需重新选择，提高使用效率

## 🔍 数据筛选

每个看板内部都有自己的筛选功能：

### 周度核心数据
- 可以选择不同的周度进行查看
- 支持近2周、3周、4周的趋势对比

### 搜索数据
- 可以选择不同的周度
- 支持按UV或PV排序
- 支持不同时间范围的留存分析

### 月度核心数据
- 可以选择不同的月份
- 支持同比环比分析

### 各模块渗透率
- 可以按月份筛选
- 可以按一级模块筛选
- 支持不同时间范围的趋势查看

## ❓ 常见问题

### Q: 看板切换后显示空白？
**A:** 首次加载某个看板时需要一些时间，请稍等片刻。如果长时间空白，请刷新页面。

### Q: 数据没有更新？
**A:** 按 `F5` 刷新页面，或按 `Ctrl + Shift + R` 强制刷新（清除缓存）。

### Q: 键盘快捷键不起作用？
**A:** 确保页面处于激活状态（点击页面任意位置），然后再使用快捷键。

### Q: 移动端显示不正常？
**A:** 建议使用现代浏览器（Chrome、Safari、Firefox等）访问，并确保浏览器版本较新。
