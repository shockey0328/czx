# 各模块渗透率看板

这是一个用于展示各模块渗透率数据的交互式看板，具有以下功能：

## 功能特性

- 📊 **多维度筛选**：支持按月份和一级模块筛选数据
- 📈 **丰富图表**：趋势折线图、排名柱状图、分类占比图、热力图
- 🎯 **KPI指标**：平均渗透率、最高渗透率、环比增长、模块总数
- 🤖 **AI分析**：集成DeepSeek AI，提供智能数据分析和优化建议
- 🎨 **现代设计**：深色主题，毛玻璃效果，响应式布局
- 📱 **移动适配**：完美支持移动端显示

## 技术栈

- HTML5 + CSS3 + JavaScript
- Chart.js 图表库
- DeepSeek AI API
- 响应式设计

## 部署说明

本项目可直接部署到静态网站托管平台（如Vercel、Netlify等）。

## 使用方法

1. 打开网页
2. 使用顶部筛选器选择月份和模块
3. 查看各种图表和KPI指标
4. 点击"生成分析报告"获取AI智能分析

## 数据说明

项目包含教育产品各模块的渗透率数据，涵盖：
- 功能模块（搜索、拍照答疑）
- 真题试卷模块（刷真题）
- 同步模块（单元卷、练习、同步学、视频、知识清单、寒暑假、特色内容）
- 备考模块（阶段复习、升学备考、推广图、学考）
