## 部署建议

### 开发环境
- 使用本地服务器（Python/Node.js）
- React应用使用开发模式

### 生产环境
1. 构建React应用: `npm run build`
2. 部署到Web服务器（Nginx/Apache）
3. 配置HTTPS
4. 启用Gzip压缩
5. 配置CDN加速

### 推荐平台
- Vercel（推荐）
- Netlify
- GitHub Pages
- 阿里云OSS
- 腾讯云COS

## 已知限制

1. **月度核心数据看板**: 需要单独运行React开发服务器
2. **浏览器兼容性**: 不支持IE浏览器
3. **数据更新**: 需要手动替换CSV文件
4. **离线使用**: 部分CDN资源需要网络连接
