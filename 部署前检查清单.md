## 🚀 部署步骤

### 1. 初始化Git仓库

```bash
git init
git add .
git commit -m "Initial commit: 数据看板"
```

### 2. 创建GitHub仓库

1. 访问 https://github.com/new
2. 创建新仓库
3. 不要初始化README、.gitignore或license

### 3. 推送到GitHub

```bash
git remote add origin https://github.com/你的用户名/你的仓库名.git
git branch -M main
git push -u origin main
```

### 4. 连接Vercel

1. 访问 https://vercel.com
2. 使用GitHub账号登录
3. 点击 "New Project"
4. 选择你的仓库
5. 配置：
   - Framework Preset: Other
   - Root Directory: ./
   - Build Command: 留空
   - Output Directory: ./
6. 点击 "Deploy"

### 5. 等待部署完成

部署完成后会得到一个URL，例如：
```
https://your-project.vercel.app
```

## 🐛 常见问题排查

### 问题：页面显示空白
- 检查浏览器控制台错误
- 检查vercel.json配置
- 检查文件路径

### 问题：数据加载失败
- 检查CSV文件是否存在
- 检查CSV文件编码（必须UTF-8）
- 检查文件名是否正确

### 问题：iframe无法加载
- 检查iframe src路径
- 检查CORS配置
- 检查子页面是否有错误

### 问题：样式错乱
- 检查CSS文件路径
- 检查CSS文件是否正确加载
- 清除浏览器缓存
