# 数据看板

一个整合了4个数据看板的统一平台，通过顶部筛选器可以快速切换不同维度和类型的数据看板。

## 🌐 在线访问

部署后的URL：`https://your-project.vercel.app`

## 📊 包含的看板

1. **核心数据看板（周度）** - 展示周度核心业务指标和趋势分析
2. **搜索数据看板（周度）** - 展示搜索功能的使用数据和用户行为分析
3. **核心数据看板（月度）** - 展示月度核心业务数据和长期趋势（需要构建）
4. **各模块渗透率看板（月度）** - 展示各功能模块的渗透率和使用情况

## 🚀 快速开始

## 🎯 使用说明

### 切换看板
1. **周度/月度切换**：点击顶部的"周度"或"月度"按钮
2. **看板选择**：使用下拉选择器选择具体的看板
3. **键盘快捷键**：
   - Alt + W：切换到周度
   - Alt + M：切换到月度
   - Alt + 左/右箭头：在当前周期内切换看板
   - Alt + 1/2/3/4：快速切换到指定看板

### 各看板功能

#### 核心数据看板（周度）
- 周度核心数据指标（活跃用户、留存率、营收等）
- B端核心数据指标
- 趋势分析图表
- AI智能数据分析

#### 搜索数据看板（周度）
- 热搜词分析（TOP 100排行、词云图）
- 搜索行为漏斗
- 搜索转化趋势
- 搜索功能留存分析

#### 核心数据看板（月度）
- 月度核心业务指标
- 长期趋势分析
- 同比环比数据对比

#### 各模块渗透率看板（月度）
- 各功能模块渗透率趋势
- 模块排名和分布
- 热力图分析
- AI智能分析和优化建议

## ⚠️ 注意事项

1. **必须通过HTTP服务器访问**，不能直接打开HTML文件
2. 确保所有子看板的资源文件（CSV、图片等）路径正确
3. 某些功能（如AI分析）需要网络连接
4. 建议使用现代浏览器（Chrome 90+、Firefox 88+、Safari 14+、Edge 90+）

## 🐛 故障排除

### 问题：看板无法加载
- 检查是否通过HTTP服务器访问
- 检查浏览器控制台是否有错误信息
- 确认子看板的路径是否正确

### 问题：切换看板时卡顿
- 清除浏览器缓存
- 检查网络连接
- 尝试使用其他浏览器

### 问题：数据不显示
- 确认CSV数据文件存在
- 检查CSV文件编码是否为UTF-8
- 查看浏览器控制台的错误信息
