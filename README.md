# 橙子学数据看板

一个完全离线的数据可视化看板系统，整合了搜索数据、核心数据和模块渗透率等多个维度的数据分析。

## 🚀 快速开始

### 1. 生成数据文件
```bash
双击运行：更新数据.bat
```

### 2. 测试验证
```bash
双击运行：完整测试.bat
```

### 3. 使用看板
```bash
双击打开：index.html
```

就这么简单！

## 📊 功能特点

- **完全离线**：无需HTTP服务器，双击即用
- **统一入口**：一个页面访问所有看板
- **自动更新**：一键脚本更新所有数据
- **智能编码**：自动检测GB2312和UTF-8编码
- **多维度分析**：
  - 周度核心数据看板
  - 周度搜索数据看板
  - 月度各模块渗透率看板
  - 月度核心数据看板（React）

## 📁 项目结构

```

└── 核心数据看板（月度）/
```

## 💡 常见问题

**Q: 更新数据后看板没变化？**  
A: 强制刷新浏览器（Ctrl+F5）

**Q: 某个看板显示空白？**  
A: 按F12查看控制台错误，运行对应的测试页面

**Q: 批处理脚本报错？**  
A: 右键选择"以管理员身份运行"

详细问题解答请查看 [最终使用指南](最终使用指南.md)
