## 📝 操作步骤

### 步骤 1：检查 Node.js

打开 PowerShell，运行：
```bash
node --version
```

如果显示版本号（如 v18.x.x），说明已安装，跳到步骤 2。

如果没有安装：
1. 访问：https://nodejs.org/
2. 下载 LTS 版本
3. 安装（一路下一步）
4. 重启 PowerShell

### 步骤 2：安装 Vercel CLI

在项目目录（`E:\橙子学数据看板\橙子学数据看板`）打开 PowerShell，运行：

```bash
npm install -g vercel
```

等待安装完成（可能需要几分钟）。

### 步骤 3：登录 Vercel

```bash
vercel login
```

会打开浏览器，选择登录方式：
- 推荐使用 GitHub 登录
- 或使用 Email 登录

完成授权后，回到命令行。

### 步骤 4：部署

```bash
vercel --prod
```

按照提示回答问题：

**Set up and deploy?**
```
Y
```

**Which scope?**
```
选择你的账号（按回车使用默认）
```

**Link to existing project?**
```
N
```

**What's your project's name?**
```
czx-dashboard
```

**In which directory is your code located?**
```
./
```

**Want to override the settings?**
```
N
```

### 步骤 5：等待部署完成

Vercel 会：
1. 上传所有文件
2. 构建项目
3. 部署到生产环境
4. 显示访问地址

完成后会显示：
```
✅  Production: https://czx-dashboard.vercel.app
```
