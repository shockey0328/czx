## 🚀 部署步骤

### 方法一：使用自动脚本

1. [ ] 双击运行 `deploy.bat`
2. [ ] 输入 GitHub 用户名
3. [ ] 输入提交信息
4. [ ] 等待推送完成
5. [ ] 访问 Vercel 网站
6. [ ] 导入 GitHub 仓库
7. [ ] 点击 Deploy
8. [ ] 等待部署完成

### 方法二：手动部署

#### GitHub 部署
```bash
1. [ ] git init
2. [ ] git remote add origin https://github.com/yourusername/czx-dashboard.git
3. [ ] git add .
4. [ ] git commit -m "Initial commit"
5. [ ] git branch -M main
6. [ ] git push -u origin main
```

#### Vercel 部署
```bash
1. [ ] 访问 vercel.com
2. [ ] GitHub 登录
3. [ ] 导入仓库
4. [ ] 配置项目
5. [ ] 点击 Deploy
```
