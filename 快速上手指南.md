# 🚀 快速上手指南

## 第一步：本地测试

### 最简单的方法（推荐）

1. **找到这个文件**: `启动本地服务器.bat`
2. **双击运行**
3. **浏览器会自动打开** http://localhost:8000
4. **开始使用！**

```
📁 橙子学数据看板/
   📄 启动本地服务器.bat  ← 双击这个文件
```

### 如果上面的方法不行

**方法A - 使用 PowerShell**:
1. 右键点击 `启动本地服务器.ps1`
2. 选择"使用 PowerShell 运行"

**方法B - 手动启动**:
1. 按 `Win + R`
2. 输入 `cmd` 回车
3. 输入以下命令：
```bash
cd "E:\橙子学数据看板\橙子学数据看板"
python -m http.server 8000
```
4. 打开浏览器访问 http://localhost:8000

## 第二步：测试功能

### 在浏览器中测试

1. **切换周度/月度**
   - 点击顶部的"周度"或"月度"按钮

2. **选择不同看板**
   - 使用下拉选择器选择：
     - 周度：核心数据、搜索数据
     - 月度：各模块渗透率

3. **使用快捷键**（可选）
   - `Alt + W`: 切换到周度
   - `Alt + M`: 切换到月度
   - `Alt + ←/→`: 切换看板

### 检查是否正常

✅ 看板能正常显示  
✅ 数据图表能加载  
✅ 切换功能正常  
✅ 没有错误提示  

## 第三步：部署到线上

### 使用 Vercel CLI（最快）

1. **安装 Vercel CLI**
```bash
npm install -g vercel
```

2. **登录 Vercel**
```bash
vercel login
```
（会打开浏览器，用 GitHub/GitLab/Email 登录）

3. **部署**
```bash
vercel --prod
```

4. **完成！**
   - 会显示一个网址，比如：https://czx-dashboard.vercel.app
   - 复制这个网址分享给团队

## 常见问题

### Q1: 双击 bat 文件没反应？
**A**: 可能没有安装 Python 或 Node.js
- 安装 Python: https://www.python.org/downloads/
- 或安装 Node.js: https://nodejs.org/

### Q2: 浏览器显示"无法访问此网站"？
**A**: 确保服务器正在运行
- 检查命令行窗口是否还开着
- 确认地址是 http://localhost:8000（不是 https）

### Q3: 看板显示空白？
**A**: 
1. 按 F12 打开开发者工具
2. 查看 Console 标签是否有错误
3. 查看 Network 标签检查文件是否加载成功

### Q4: 无法部署到 Vercel？
**A**: 
1. 确认已安装 Node.js
2. 确认网络连接正常
3. 尝试重新登录：`vercel logout` 然后 `vercel login`
