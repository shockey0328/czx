## 文件说明

### 根目录文件

#### index.html
统一看板的入口页面，包含：
- 顶部导航栏（Logo、标题）
- 周度/月度切换按钮
- 看板类型选择下拉菜单
- 看板容器（用于加载子看板）

#### main.js
主要业务逻辑，包含：
- 看板配置信息
- 周度/月度切换逻辑
- 看板加载逻辑
- iframe管理
- React应用特殊处理

#### styles.css
统一看板的样式文件，包含：
- 全局样式
- 导航栏样式
- 按钮和选择器样式
- 加载动画样式
- 响应式设计

#### 启动看板.bat
Windows批处理脚本，用于：
- 一键启动主看板服务器
- 一键启动React应用服务器
- 自动打开浏览器

### 子看板目录

#### 核心数据看板（周度）
- **技术栈**：HTML + CSS + JavaScript + Chart.js
- **特点**：四宫格布局，包含核心指标、趋势分析、B端数据、AI分析
- **数据文件**：周度核心数据.csv、B端核心数据.csv等

#### 搜索数据看板（周度）
- **技术栈**：HTML + CSS + JavaScript + ECharts
- **特点**：热搜词分析、词云图、漏斗图、留存热力图
- **数据文件**：第X周搜索词.csv、搜索行为漏斗.csv等

#### 核心数据看板（月度）
- **技术栈**：React + Recharts + TailwindCSS
- **特点**：组件化设计，交互丰富，AI智能分析
- **数据文件**：月度核心数据.csv、B端核心数据.csv
- **运行方式**：需要npm start启动开发服务器

#### 各模块渗透率看板（月度）
- **技术栈**：HTML + CSS + JavaScript + Chart.js
- **特点**：趋势图、排名图、热力图、数据表格、AI分析
- **数据文件**：各模块渗透率.csv

## 数据流程

1. **用户访问** → 打开 index.html
2. **选择维度** → 点击"周度"或"月度"
3. **选择类型** → 从下拉菜单选择具体看板
4. **加载看板** → main.js根据配置加载对应看板
5. **展示数据** → 子看板读取CSV文件并渲染图表

## 技术架构

### 统一看板层
- **框架**：原生HTML/CSS/JavaScript
- **加载方式**：iframe嵌入
- **优点**：简单、独立、易维护

### 子看板层
- **静态看板**：直接通过iframe加载HTML文件
- **React应用**：通过iframe加载localhost:3000

### 数据层
- **格式**：CSV文件
- **读取**：各看板使用PapaParse或原生方法解析
- **更新**：替换CSV文件后刷新页面

## 扩展说明

### 添加新看板

1. 在对应的周度/月度目录下创建新看板
2. 在 main.js 的 dashboardConfig 中添加配置：
```javascript
'new-dashboard': {
    name: '新看板名称',
    path: '新看板目录/index.html',
    type: 'static'  // 或 'react'
}
```

### 修改样式

- 统一看板样式：修改 styles.css
- 子看板样式：修改各子看板目录下的样式文件

### 更新数据

直接替换对应目录下的CSV文件，刷新页面即可。

## 部署说明

### 本地部署
按照"使用指南.md"中的说明启动服务即可。

### 生产部署
1. 构建React应用：
```bash
cd 核心数据看板（月度）
npm run build
```

2. 将所有文件部署到Web服务器

3. 配置服务器支持：
   - 静态文件服务
   - SPA路由（React应用）
   - CORS（如果需要）

### 推荐部署平台
- Vercel（推荐，支持React）
- Netlify
- GitHub Pages
- 自建Nginx服务器

## 维护建议

1. **定期更新数据**：建议每周/月更新CSV文件
2. **检查依赖**：定期更新npm依赖包
3. **浏览器测试**：在主流浏览器中测试功能
4. **性能优化**：监控加载速度，优化大文件
5. **备份数据**：定期备份CSV数据文件

## 常见修改场景

### 修改Logo
替换根目录和各子目录下的 logo.png 文件

### 修改主题色
在各看板的CSS文件中搜索 #FF6B35（橙色）并替换

### 添加新指标
1. 更新CSV文件添加新列
2. 修改对应看板的JavaScript代码读取新数据
3. 更新图表配置显示新指标

### 调整布局
修改对应看板的CSS Grid或Flexbox配置
