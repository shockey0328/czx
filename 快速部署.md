## 🚀 三步部署

### 第 1 步：使用 GitHub Desktop（最简单）

1. **下载 GitHub Desktop**
   - 访问：https://desktop.github.com/
   - 下载并安装

2. **添加项目**
   - 打开 GitHub Desktop
   - File → Add Local Repository
   - 选择你的项目文件夹

3. **发布到 GitHub**
   - 点击 "Publish repository"
   - 仓库名：`czx-dashboard`
   - 取消勾选 "Keep this code private"（如果想公开）
   - 点击 "Publish repository"

✅ 完成！代码已上传到 GitHub

---

### 第 2 步：部署到 Vercel

1. **访问 Vercel**
   - 打开：https://vercel.com
   - 使用 GitHub 账号登录

2. **导入项目**
   - 点击 "Add New..." → "Project"
   - 找到并选择 `czx-dashboard`
   - 点击 "Import"

3. **部署**
   - 保持所有默认设置
   - 点击 "Deploy"
   - 等待 1-2 分钟

✅ 完成！你的看板已上线

---

### 第 3 步：访问你的看板

部署完成后，Vercel 会给你一个地址：

```
https://czx-dashboard.vercel.app
```

或者你可以配置自定义域名。

## ❓ 遇到问题？

### 如果 GitHub Desktop 推送失败

使用命令行：

```bash
# 1. 设置远程仓库（替换 YOUR_USERNAME）
git remote set-url origin https://github.com/YOUR_USERNAME/czx-dashboard.git

# 2. 推送
git push -u origin main
```

### 如果 Vercel 部署失败

1. 检查部署日志
2. 确认 GitHub 仓库可访问
3. 重新触发部署
