# 看板标题优化说明

## ✅ 优化完成

已成功移除各子看板内部的logo，只保留标题文字。

## 🎯 优化目标

由于主看板顶部已经有统一的logo和导航，各个子看板内部不需要重复显示logo，这样可以：

- ✓ 避免视觉重复，界面更简洁
- ✓ 节省顶部空间，内容区域更大
- ✓ 统一设计风格，层级更清晰
- ✓ 提升用户体验，减少干扰

## 🔧 技术改进

### HTML结构简化

**优化前：**
```html
<div class="navbar">
    <div class="logo">
        <img src="logo.png" alt="Logo">
        <div class="logo-text">标题</div>
    </div>
    <select>...</select>
</div>
```

**优化后：**
```html
<div class="navbar">
    <div class="logo-text">标题</div>
    <select>...</select>
</div>
```

### CSS样式清理

移除的样式：
- `.logo` 容器样式
- `.logo img` 图片样式
- `.header-left` 布局样式
- `.logo-container` 相关样式
- `.logo-fallback` 后备样式

保留的样式：
- `.logo-text` 标题样式
- 优化字重为 600

### 统一标题样式

所有看板标题统一为：
- 字体大小：24px
- 字体颜色：#FF6B35（橙色）
- 字重：600（半粗体）
- 对齐方式：左对齐

**简洁的设计原则**

- 保持一致：所有看板统一样式
