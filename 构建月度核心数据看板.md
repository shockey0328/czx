# 构建月度核心数据看板

月度核心数据看板是一个React应用，需要先构建才能使用。

## 🔧 构建步骤

### 1. 进入月度核心数据看板目录

```bash
cd "核心数据看板（月度）"
```

### 2. 安装依赖（如果还没安装）

```bash
npm install
```

### 3. 构建应用

```bash
npm run build
```

构建完成后，会在当前目录生成一个 `build` 文件夹。

### 4. 更新dashboard.html

构建完成后，需要修改 `dashboard.html` 中月度核心数据看板的路径：

将：
```html
<iframe src="核心数据看板（月度）/public/index.html" ...>
```

改为：
```html
<iframe src="核心数据看板（月度）/build/index.html" ...>
```

## 🚀 快速方案（推荐）

如果不想构建React应用，可以使用开发服务器：

### 方案A：使用React开发服务器

1. 进入目录：
```bash
cd "核心数据看板（月度）"
```

2. 启动开发服务器：
```bash
npm start
```

3. 服务器会在 http://localhost:3000 启动

4. 修改 `dashboard.html`，将月度核心数据看板的iframe src改为：
```html
<iframe src="http://localhost:3000" ...>
```

### 方案B：创建简化版（最简单）

我可以为你创建一个不依赖React的简化版月度核心数据看板，直接使用HTML+CSS+JS。

## ⚠️ 注意事项

- React应用需要Node.js环境
- 构建过程可能需要几分钟
- 构建后的文件会比较大（包含所有依赖）
