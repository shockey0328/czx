# 更新日志

本文档记录项目的所有重要变更。

## [2.0.0] - 2025-02-06

### 新增功能 ✨
- **小橙子AI助手**: 全新的智能数据分析助手
  - 固定在页面左侧，点击即可对话
  - 基于DeepSeek AI提供智能分析
  - 支持自然语言问答
  - 实时基于当前数据进行分析

- **AI记忆与学习能力** 🧠
  - 对话历史记忆（保存最近20条）
  - 用户偏好学习（自动识别关注指标）
  - 数据洞察积累（保存最近50条）
  - 个性化欢迎消息

- **智能上下文理解** 💡
  - 结合历史对话提供更精准回答
  - 关联同月份的历史分析
  - 基于用户偏好优化回答重点

### 技术改进 🔧
- 使用LocalStorage实现数据持久化
- 优化AI提示词构建逻辑
- 改进错误处理机制
- 添加调试日志输出

### 文档更新 📝
- 更新README.md，添加小橙子功能说明
- 新增《小橙子AI助手技术文档.md》
- 新增《小橙子助手使用说明.md》
- 更新项目结构说明

### 样式优化 🎨
- 新增小橙子专属样式文件
- 优化对话窗口UI设计
- 添加打字指示器动画
- 改进响应式布局

## [1.0.0] - 2025-01-XX

### 初始版本 🎉

#### 核心功能
- **数据看板**: 展示月度核心数据
  - 用户模块（月活、留存）
  - 收入模块（营收、订单、ARPU、ARPPU）
  - 使用模块（深度访问率、使用率、会员活跃率）

- **趋势分析**: 
  - 4个核心指标趋势图
  - 支持3/6/12个月时间范围
  - 交互式图表展示

- **AI数据分析**:
  - 基于DeepSeek AI的智能分析
  - 关键变化分析
  - 增长趋势分析
  - 策略建议

- **B端数据展示**:
  - 签约、开票、到款金额
  - 订单数、新签、续签统计
  - 关键比率分析

#### 技术栈
- React 18
- Recharts 图表库
- PapaParse CSV解析
- Axios HTTP客户端
- DeepSeek AI API

#### 设计特点
- 橙色主题设计
- 响应式布局
- 现代化UI组件
- 环比同比计算

### 版本号规则
遵循语义化版本 (Semantic Versioning):
- **主版本号**: 不兼容的API修改

- **修订号**: 向下兼容的问题修正
