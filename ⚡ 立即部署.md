# ⚡ 立即部署到 Vercel

由于 GitHub 仓库问题，我们直接使用 Vercel CLI 部署，跳过 GitHub。

---

## 🚀 三步部署

### 第 1 步：安装 Vercel CLI

打开命令行（CMD 或 PowerShell），输入：

```bash
npm install -g vercel
```

**如果提示 npm 不存在**：
- 先安装 Node.js: https://nodejs.org/
- 下载并安装后，重新打开命令行

---

### 第 2 步：登录 Vercel

```bash
vercel login
```

会打开浏览器，选择登录方式：
- GitHub
- GitLab  
- Bitbucket
- Email

**推荐使用 Email**（最简单）

---

### 第 3 步：部署

在项目目录中执行：

```bash
cd "E:\橙子学数据看板\橙子学数据看板"
vercel --prod
```

## ❓ 常见问题

### Q: npm 命令不存在？
**A**: 需要先安装 Node.js
1. 访问 https://nodejs.org/
2. 下载 LTS 版本（推荐）
3. 安装后重启命令行

### Q: vercel login 没反应？
**A**: 
1. 检查浏览器是否被拦截
2. 手动打开 https://vercel.com/login
3. 登录后回到命令行

### Q: 部署失败？
**A**: 
1. 检查网络连接
2. 确认在正确的目录
3. 尝试 `vercel logout` 然后重新登录

### Q: 想修改项目名称？
**A**: 
1. 访问 https://vercel.com/dashboard
2. 找到项目 → Settings → General
3. 修改 Project Name
